# OneCash Clone – تطبيق إدارة محفظة وماليات

## نظرة عامة
تطبيق مالي للهاتف يتيح إدارة المحفظة وتحويل الأموال وسحب/إيداع (تصميم مشابه لتطبيق OneCash).

### الميزات الأساسية
- تسجيل ودخول للمستخدمين
- إدارة المحفظة (رصيد، شحن، سحب)
- تحويل مالي بين المستخدمين
- سجل معاملات واضح
- دعم دفع بوسائل متعددة (بطاقات، تحويل، إلخ)
- واجهة مبسطة وسهلة الاستخدام

## التشغيل السريع
1. شغل الخادم:
   npm install && npm run dev
2. شغل التطبيق المحمول:
   npm install && npx expo start

> ستحتاج لتعديل المتغيرات في ملف .env لكل من النطاقين حسب مفاتيح API وقاعدة البيانات

## التقنيات
- Backend: Node.js, Express, MongoDB
- Frontend: React Native, Expo
- الدفع: Stripe, PayPal
